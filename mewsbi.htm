Updates

05.02.23

-* hi! i hope you are enjoying the new UCs! please see the new tentative policy regarding UC wearables below:

*- cats wearing a NC UC Style are eligible to be listed as the colour they appear as, even if the base colour doesn't match. eg; a blue aisha wearing a grey UC wearable can be listed as grey. if the colour/skin changes, the cat will be delisted, which is the same rule i follow for colour changes currently. you are of course welcome to contact me if you've swapped a UC skin around and would like to stay listed!

12.10.23

-* new rule alert! all listed kitties must now be wearing a background. all backgroundless kittens have been unlisted, and over 130 listings have been removed. As the directory grows, this change makes it easier for me to manage, and see when pets have been traded. i don't want to list pets without their owner's permissions, so i hope you will understand.
-* all imposters/colour changes/species changes removed
