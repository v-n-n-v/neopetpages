Leave No Name Behind!

A list of LGBTQIA+ names stuck on baduser.

About
On December 18th, 2020 (Y22), the ban on LGBT+ topics was lifted. Staff started allowing users to request names related to those topics be released from baduser, the staff account where pets with inappropriate names are sent. While many recognizable terms and other very well named pets were released shortly after, countless others remain stuck. This page aims to shine a spotlight on some of the more difficult to find pets that might never have a chance of being released otherwise.

As much as I wish it were, this list is not comprehensive. There are probably thousands of other names stuck on baduser as well.

Pets that are still on baduser will have a visible image, but their petpage will say Oops! No Neopet with this name exists in Neopia! If the pet's petpage is visible, it means that they have already been released for adoption. (Yay!)


Interested in one of the pets listed?
If you would like to adopt one of the pets listed here, you will have to send a ticket requesting them to be released. Pets that are released are moved into the pound so that anyone will have a chance to adopt them.

Please don't request too many names at once! Support has to move every pet manually, which can take 1-3 minutes per pet.

Some pets may not be able to be released if staff finds that there are other issues. In particular, Lutari cannot normally be placed into the pound. Unfortunately, I'm not sure if anything can be done for these pets at this time. ):
