Apoph's untaken Pokémon misspell page

Hello! Have you ever wanted a certain Pokémon name (or just a Pokémon name in general), and all the Pokémon you like are not UFT or you don't have the right offer? Well, I decided to look up a bunch of untaken Pokémon misspells via tilesetting and compile this page. Nothing fancy, but I believe it gets the job done. I'm going to try to keep this page updated, but if you take a name and want it off the list, feel free to neomail me here. Without further ado, here's the list, complete with pet images so you can see if the name is taken or not! I'll try to update it with new names over time. NOTE: Pet images aren't working at the moment, so I've taken away the images and replaced them with links. I do check every now and then to see which names are taken, but I don't check every day so there may be some that have been taken since I updated the list.
