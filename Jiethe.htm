We'll be counting stars.

Hello there, and welcome to Syzygy, my premades layout site/layout dump! I'm Emily (emilyhunter5034), and I mainly run Skyfall, a graphics site. I code plenty of layouts for my own use and for fun, and it seemed a bit of a waste to abandon my old codes. That's why I've decided to set up this site so all my codes can be up for grabs! Quick navigation is on the upper right corner, hover over the triangle and you'll see it!

Before using a layout, I recommend reading the notes beside it; it's especially helpful if you'd like to customize your layout! To copy code, ctrl + a then ctrl + c. If you want a full preview, drag & drop the image to the address bar. If you have any questions, suggestions, or feedback, let me know!

requests

In hopes of seeing more community sites, petpage layout requests are now open to anyone looking to start one, or is already running a site! Please note that at least some content must be available when you're requesting a layout.

Any site that directly gives back to the community as opposed to personal projects like portfolios and applications are considered as a community site. This includes layouts sites such as Syzygy, requests or graphics sites like Skyfall, font sites and site directories. Of course, if you own a layouts site, you probably don't need to request from me ;)
