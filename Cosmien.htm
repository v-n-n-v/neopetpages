using the maker

• Browse through the categories in the navigation on the left-hand side of the page or just scroll through and click on whichever feature you want to add
• There are multiple categories for some features so you can have 2 different patterns, multiple rings, etc
• If you want to save the image afterwards you can take a screenshot and crop it in another program, like Paint, or use the Windows snipping tool ^u^

rules & requests

• Please only use this for Neopets and not off-site
• Link your image back to this page if you use the maker (you can also just put credit anywhere else like at the end of your petpage, use a link back button, etc)
• Do not claim the images as your own or use them as "adoptables" off-site -- idk if anyone would even do this but I guess I should say it anyway?
• If you want a transparent background, you can mail me with the image and I can remake it for you c:
• Feel free to also request more colours/patterns/etc, my artistic abilities are limited so I can't make anything crazy but I'd love to add more stuff!
• The "custom" heading is for requests or things I decide to add myself that don't fit in the existing categories
