12.21.19 - Recently, I dug up some old files that contained Praedius' original petpage as it was coded in 2003. The last update to it was in January of 2005. It was a tangled mess, copied and pasted from "how to" guides all across the web, smashed together in an amalgamation of tags that somehow, miraculously, still managed to make a more or less functional petpage. I've learned quite a bit about html and css since then, so I've completely recoded his page from scratch, replacing all the tables with divs and actually having the ability to write the code myself rather than reverse engineering premades, dissecting code snagged from various tutorials, or having Microsoft Frontpage 2000 write all the html for me. I've tried to leave a lot of the original text intact, preserving Praedius' "voice" as he was written all those years ago. I no longer have my teenage angst to guide me in writing for him, so perhaps it is more authentic this way. I'll have his old petpage, in its original state, uploaded to another petpage once this one is complete so folks can draw a comparison. We've come a long way! Thanks for coming on this journey with us!
~Arttimo
