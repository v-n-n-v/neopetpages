WELCOME

Hello and welcome to Hygge, a graphic resources & requests site brought to you by Minyoung (minyoundwind). Before using the images on Hygge, please read through the terms of use in this page and always give a link back credit to Hygge, as huge amount of efforts and time was put into this site. Feel free to reach out to me via neomail if you have any question, comment or suggestion :)

REQUEST

General Guidelines

Hygge offers three different types of requests (i.e. PNGs, floats and backgrounds) and re-sizing request. Below you'll find the general guidelines and rules. Each request type has its own guidelines as well. Be sure to check out both.

  • I only do neo-related images. You can find those on Neopets, JellyNeo, DTI, Dr. Sloth's Image Emporium, etc.
  • Credit is required. You must use a clickable link back.
  • Do NOT claim my work as your own. Do NOT use the requested image(s) in any spotlight or contest submission.
  • Please make sure you have enough space to receive neomails.
  • You'll receive a confirmation message from me within 2 days. If you do not receive the message by then, please do re-submit your request again because sometimes neomail system may be broken.
  • It may take me anywhere from one day to two weeks to complete your request (depending on the type of request). In the confirmation message, I'll let you know when your request is expected to be ready.
  • Please only submit ONE request each time. You may submit another request when the current one is completed and if the status is still open.

PNG REQUEST

Please read through the following guidelines below before requesting, and send a neomail to me with the form.



  • If the image you request for is not a wearable item , I'll re-draw the image. I reserve the right to decline a request if the one in question is too difficult for me to re-draw.
  • The smallest image I can re-draw is 80px X 80px- for instance, a NP item. You can find those images at JellyNeo item database.
  • For wearable items, you can choose the item(s) or element(s) you like and then I'll re-arrange them into one PNG. An example of this is the top banner of Hygge site, where I use elements from 3 different wearable items to create the banner. Please visit float request page to understand the way I work with wearable items (i.e. SWFs).
  • For PNG renders, delivery time depends on how detail an image is. For instance, the more linings and color gradients the image has, the longer time it may take for me to complete the request.
  • Just like the premade PNGs, requested PNGs usually come in 3 different sizes for you to use.
  • I reserve the right to determine the requested PNG may or may not be added into the premade section.

BACKGROUND REQUEST

Please read through the following guidelines, and send a neomail to me with the form.

  • You can use JellyNeo's wardrobe or Dress to Impress (DTI) to search and view wearable background.
  • The background will be 2000px X 1600px by default. If you need other size, please specify it in your request.
  • Usually your request will be completed within two day.
  • I reserve the right to determine the requested background may or may not be added into the premade section.


RE-SIZING REQUEST

Please read through the following guidelines, and send a neomail to me with the form.

  • Both premade and custom images at Hygge can be re-sized.
  • Please specify either width or height you want because the aspect ratio of an image must be kept the same to avoid distortion.
  • Usually your request will be completed within 2 days.


PNGs

  • Hygge's PNGs are either re-drawn using a vector based program (PNG Renders), or created from SWF files (Wearable PNGs).
  • The PNGs come in three different sizes for your convenience. For best results, simply choose the nearest size you're looking for.
  • All the PNGs can also be used as floats. Simply save the PNG and crop out the size you want (as mentioned above there are three sizes available), and then re-upload to a image host website to use.
  • If you need a specific size for a PNG, you can send in a re-sizing request. I'm happy to resize the PNG for you.

    FLOATS

    • Below you'll find Hygge's float collections such as headers, footers and sides, offered by me for you to use in your layouts or graphics.
    • Different from the regular PNGs, floats are PNG images particularly designed to fit various CSS layout (such as user lookup, petpages, shops or galleries). The size of each float image is carefully chosen so that you can better manage the positions of navigation links in your layout.
    • If you have ever used floats in layouts before, you might notice that some links in your layout become unable to click. That is because we use z-index code for floats and thus floats would block links underneath them. You can fix this by adjusting the position of the float or the position of the links.
      It's also good to keep in mind that for mobile screens, some floats such as sides would probably block more links/navigation than PC screens in layouts like user lookups.
    • Just a note to explain why I do not use html code like (img src="image.png" style="top: 0px; left: 0px; height: 100%; width: auto; z-index:1;position: fixed;") for side floats. It is because by setting height: 100%, the image would get re-sized depending on viewer's screen size, making side floats hard to design and image quality would go down as well. Also for mobile screens, floats would not look good when setting height: 100%.

    • If you're looking for a larger or smaller size of a specific PNG float, please send in a re-sizing request.
