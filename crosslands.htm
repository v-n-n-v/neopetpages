Rules

1. You may request up to two buttons at a time. If you do, please put the requests in the same form.

4. If your request doesn't receive a reply after two days, please resend your request.

5. Please wait a week before requesting again.

Tips and Tricks

1. If you are using a pixel font, make sure you set the alias to "None". If you leave your pixel font aliased, it will be blurry.

2. Consider using Smart Sharpen or just the Sharpen Tool if your focal image is blurry.

3. For the Shine animation, set your animation layer to "Overlay" and adjust the opacity for a more subtle effect.

4. Don't be afraid to mix up fonts in a button. Try to keep it to two fonts maximum. The best combination is generally a pixel font used in conjunction with another font, usually a script font.

5. Set the white inner border of your button to "Overlay" or "Soft Light" rather than "Normal" so that the border is less stark. The border is meant to bring a button together and highlight the text and focal image, not to be the center of attention.
