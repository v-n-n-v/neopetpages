Getting Listed

Listing Information

To list a pet you will need the following information:
Neopet's name (on site), link to where more information is (pet page or look up), and a button (50 x 50 or 88 x 31)
- Character hubs are accepted but you must be able to link to individual sections per character if you want individual listings. Otherwise it will be added to the Hub section

- Only characters based on a Neopet you own are accepted at the moment.

- If you don't have a button for your pet, I can make a placeholder, but I highly suggest finding a button request site if you can't make one yourself!
check somewhere like Unwound Clock or Bookmarks for button request pages!

Remove or Update a Listing?

If you wish to remove a listed character, or update a link or button, please feel free to send me a neomail!

Periodically I will go through listed character to make sure pages are still active. If a page is down, or significantly nonfunctional (broken code or images), I will remove the link and send you a neomail. If you fix the page and wish to have it added back, you will have to fill out the above form again. Sorry for any inconvenience.
