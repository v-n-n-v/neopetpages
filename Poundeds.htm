You may be wondering what does that even mean anyways?

The first thing to take note of is that not every letter of the aphabet is listed in that list. Which means that pet's with names that start with the letter not listed (E, I, O, N ....) those pets will not get stuck in the pound.

Lets Quote Megablade for this section, they explain it better then I can:

Basically, pet names are grouped by the first two letters of the name. We call this a "string." Only certain "strings" of names will stick, and not every name in a given string will. The three letters shown above indicate where in the string the names begin to stick."

Example: Lets use the begining string of SEE- & take not of the SHA- String also.

SI- is the first string listed that starts with the letter S. Name such as Sidney would not stick as it has not met both of the parameters for sticking. Yes Sidney does Start with SI but if you look at where the SI string starts it starts at the letter L (SIL-) So any names that are alphabetically before SIL will not get stuck in the pound if they have been abandoned. Names Such as Silas, Simon, Sinker ... would.
