User Guide

Hiya! Recently, Absolutely has been revamped with several new features to improve site quality and ease of use. However, not all of these changes are obvious.

To start off, if you would like to access the premades you'll need to read the rules first. After you've done that, you can click the premades button. Go ahead, do it now! You won't navigate away from this page.

Do you notice anything different? Instead of being redirected to the premades page, the update sidebar has been replaced with two sublinks - Image and CSS. To get to the layouts, click either link and you'll be taken there.

If you would like to go back to the updates, simply click the back button beneath the links. If you'd like to read more about the updates, click the more button to be taken to the new blog feature of this site!

This was just a basic rundown of what's new with Absolutely. If you have further questions, please neomail me.


rules

The rules for using my premades and resources are fairly simple - I only have two, but you have to follow them!

Leave the credit on.
But I'm gonna change - Doesn't matter.
It won't even look like - Not an excuse.
You must leave the credit on, even if you edit my layouts to the point that you're only using them as a base code. If I see or hear of anyone stealing my codes, I'll contact you with a fair warning before moving on to harsher consequences.

Do not redistribute my codes.
There are no rebuttles for this one. Just don't do it. It's not fair for me to have my codes available for other people's use if they're just going to claim them as their own.

These rules may sound harsh, but they're necessary in order for me to keep providing you with premades and resources. Now that we've got that out of the way, why don't you check out the premades?

frequently asked questions

Below are my most frequently asked questions! If you don't see your question here, feel free to neomail me to get it answered.

Q: I can't find the save changes button!
A: No problem! To find the save changes button, you'll need to zoom out in your browser until it's visible. This is usually done by pressing "ctrl" and "-" at the same time.

Q: Do you do custom layouts?
A: Not formally. If you're interested, please neomail me to see if we can work something out.

Q: Why didn't you accept my layout request?
A: Either because it wasn't feasible or because I wouldn't have enough time to make it to the best of my ability. Keep in mind that this is a premades site, not a request site. If you'd like a custom layout, check Cosmopoltian or Seraphic.

Q: You said that you'd complete my request but then you said that you couldn't.
A: See above.

I'll add more questions as I get them, but that's it for now!
