      Welcome

      Whale-come to Whale.CSS; a premade CSS layout site run by me, Thomas. I aim to provide you with quality CSS layouts that are good enough to just put content on and look good, yet editable enough to make it your own. Here at Whale.CSS, you can get premade pet pages, user lookups, and pet lookups. All layouts are almost purely CSS. The only thing other than CSS is the occasional banner or link back button.

      You might be thinking to yourself: Why do I want a CSS layout then? Well, a CSS layout loads at a much quicker pace. This is because there are fewer graphics to load, which take up a lot of space. Also with a CSS layout, you have a lot more room for editing. With a graphic layout, what you see is what you get; there aren't a whole lot of customization opportunities. With a CSS layout, however, you can basically customize everything to fit your needs!

      History


      Whale.CSS was created when I realized how much fun it is to code CSS layouts! Up until that point, all I had done, was made graphical layouts. I personally like making graphic layouts, but CSS layouts are just so much more versatile. Anyways, the idea popped into my head on March 8, 2014, and construction began on that same day.

      Before I could begin the site, I had to think to myself: what should I call my site? I started looking at other CSS sites, but most of them were *insert name and add apostrophe "s"* CSS. I wanted something a little different; something to make my site unique. From previous knowledge, I knew that each CSS layout designer has a unique code. They name their .div boxes in ways that they know they are theirs, and use certain filler text too. So I got to thinking about that in hopes it would help me name my site. Then, it hit me--whales!

      If you don't know me well, then you probably don't know about my bizarre obsession with whales. But now that you do, this will all make more sense! I figured I could name .div elements after my favorite whales/whale terms. I could also make my filler text about whales too (to make my site unique). So what name is more fitting than Whale.CSS. That was the first name that came to mind, and it stuck!
