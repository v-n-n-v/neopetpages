How to Read These Charts:
Check the averages to the left. This is what to expect based on the data:

Less than 3.0 / Green / Often auto-trophy, occasionally auto-gold.
3.1 - 6.0 / Yellow / Occasionally auto-trophy, usually will need advertising for gold.
More than 6.0 / Red / Popular category, will almost always need advertising for gold.

There are always outlier weeks!



In order to obtain the Beauty Contest avatar, you only need to place in species.
Therefore, any green-averaged species is recommended.
