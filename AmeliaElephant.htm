You should be able to find this code for any colour you use somewhere in your art program that you used to design your lookup. Have a look with the colour picker and see if you can spot it. Adding More Text Boxes

Have a lot to say, or want to display different information in different places throughout your userlookup? Maybe a wishlist in one, and your side accounts in another? No problem at all! Here's how to easily add in more.

First, copy and paste this code somewhere between your style tags - at the bottom under the #aboutbox and #customnav sections would be good.

Next, you need to change #aboutbox to something different. You could even just add a number to the end. For example you could use #aboutbox2 or #wishlist. This will allow the code to put your HTML in the right place.

Next, edit the placement using the same method as moving the modules above, by changing the top and left values. You can also change the size of the box to match your designated area by changing the width and height aspects.
