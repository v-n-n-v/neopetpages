css layouts by jewlz

Of course, all our layouts are customizable. Element was designed to be used with the jewel family sites to create the perfect layout. All content is interchangeable within sites.     some quick notes about our layouts


    Element layouts have multiple views to show exactly what kind of special coding is within and how the layout looks. Hover over each layout section to display more information about the layout including layout difficulty. You can find the layout code in the boxes below the images. Use CTRL A + CTRL C to copy all the text. As previously stated, all of these layouts can be easily edited. There is also additional instructions for use within the layout coding itself.

    image quality: Some of the GIFs of the layout previews may have a bad image quality. It's just from uploading such a large animated image. The layout itself will have 100% quality images. No worries there!

    can't find the save button? Layouts that have a 'hidden' save button usually just need to be zoomed out. Try using CTRL - a few times until you see the button.

    still not there? Try holding shift then hit the tab / indent button twice. ( I suggest saving a copy of your work in a notepad first before doing this. Just in case. )

    Looking for a different color scheme? Try looking here for a large assortment of color palettes!
