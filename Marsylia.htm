Pet display CSS codes

Default coding on userlookups only displays up to 5 pets, with the 5th one being slightly cut by 10px.
The CSS on this page will help you edit that if you have no CSS/HTML knowledge. Just copy&paste the codes at the beginning of your lookup code.
These codes also disable the scrolling on pets.

Please note if you have any kind of custom CSS that edits the look of pet images and/or changes the total width of your lookup, there's a high chance the following codes won't work properly without adjustments

Very long petnames fix

If you happen to have one or more very long petnames and they end up overlapping, you can simply add the following line before the ‹/style› tag. It reduces the size of the petnames so that they don't overlap.
If your petnames overlap but aren't too long, you can try using 7.5pt instead of 7pt. If you have VERY long petnames and 7pt isn't small enough, you can reduce the number to 6pt.
