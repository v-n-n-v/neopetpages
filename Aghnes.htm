{Nix:} an Untaken Names Page

Hello, and welcome to Nix, an untaken names page run by Saint. Established in January of 2019, Nix features both creatively made-up names and compound names since each owner has their own preference of style and taste when it comes to naming their Neopets.

Nix uses alphabetical ordering when listing pets to make it easier on the owner. However, it is also perfectly fine to skim the lists if the owner doesn't yet have a preference on what their new pet's name should start with.

Sometimes, Saint will allow requests on making up names if the owner has a certain preference for a name but does not see a desirable one listed on Nix. Whether or not Saint is open to name requests is always going to be listed below.

If you are interested in looking for a name on Nix, please familiarize yourself with the rules before picking one. It is only through cooperation that Nix is able to thrive, and if the rules are followed, then Nix can provide more untaken names to owners who may need them.

Rules


1) If you take a name, please Neomail Saint to tell her that you've taken it so that she can take it off the list. Ideally, sweeps are done once every Sunday, but it makes it easier for everyone if the taken name is erased from the page as soon as it's claimed.


2) Please do not take credit for any of the names posted on Nix. For instance, if there is an untaken names board on the Pound Chat, please do not visit Nix and copy-paste the name lists to post on the board, as it may result in people taking names and not telling Saint to take them off the list. Instead, post the link to Nix so that names can be properly taken care of!


3) For requests, you may add as little or as much detail as you'd like the name to have, but please have at least one boundary for Saint to work with, whether it be a starting letter, a consonant-vowel format, a certain number of letters, whether the name should be masculine or feminine, etc.


4) Please use the following form when making requests for an untaken name. If you have no preference in a certain section, it is fine to just put "N/A" as long as one section is filled out. Title your Neomail "Untaken Name Request" so that Saint knows it's Nix-related business and can answer it quickly!

5) This is not really a rule as much as it is for fun, but if you finish the paint, morph, or customization of a pet whose name you originally took from Nix, please feel free to Neomail Saint a picture of your pet! It is always a joy to see the creativity of others.
