he Site & Coding Help Board on the Help Chat aims to help with any site-related problems, whether you're trying to find a userlookup layout, create a graphic, or even set up a site of your own! Board replies typically take 1-2 days. Though not required, please check back to let us know if our replies were helpful to you, or even stay to chat!

It is highly recommended to read the FAQs below if you have a question; chances are it has already been answered before. However, if you're unsure, feel free to ask away; we don't bite!

FAQ

How do I display pets in rows on my user lookup?
Try using the code from Code Snippets. If you are using a premade layout, you may need to adjust the code accordingly. Ask on the board if you are unsure how to do so! However, if you're only looking to show pets in rows on the default lookup, use the code from this page.

Where can I find a premade userlookup / petlookup / petpage layout?
There are many sites that offer them in various different styles, so try checking a directory! Kalux Directory and Quality Petpages are some of the most recently updated directories.

Is anyone taking requests for custom graphics / fonts / layouts?
Unwound Clock lists most active request sites, and if their requests are currently open. Do note when the site was last updated; some sites may have changed their request status in between updates!

Where can I find some premade fonts/neoboard signatures?
The Font Directory lists pretty much all font sites with usable content, as well as whether they offer text-based or font-based signatures and whether they take requests. They also have links to font-making guides and resources.

How can I display a larger pet image on my pet lookup?
This page has the code to show your pets in 500x500. If you would like a different size, you can access your pet's image by using this url: https://pets.neopets.com/cpn/PET_NAME/1/5.png; simply change PET_NAME to your pet's name, and change the number 5 to anything from 1-7 for various different sizes.
