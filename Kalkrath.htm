* in the css shorthand with four values, they run clockwise starting with the value for top.

The provided code gives you 3 rows of links in the link area plus a top 'home button' row. If you need more or less space, even if it's just because you changed the font, you will need to adjust the following attributes:
1) The height in the .deluxe_magic_window class
this is how much height this whole box monstrocity takes up on the page.
2) The margin-top in the .deluxe_magic_window class
(keep it negative, this moves the window box upward on the page overlapping the links, when you click on a link the top of magic window lines up with the tippy top of your screen! if you have a fixed navigation bar at the top of your page, you'll want to take that into account and shift window box upwards more.)
3) The padding-top in the .deluxe_interior_anchor class
This bit makes sure that the text hits below the link area and not underneath it. too high and it's too far from the links, too low and the links will cover it up.

* unfortunately picking the numbers means doing some math, so just... fire up the 'inspect element' tool on your fav browser and play around with it. The design of it should give you some wiggle room so it's okay if things don't line up ~perfectly~.

*** that being said, these are some numbers that work out (assuming a row of links is roughly 30 pixels tall, and you want roughly 30 pixels between the link section and the text section):

2 rows of links (plus home button): .deluxe_magic_window height: 520px, .deluxe_magic_window margin-top: -210px, .deluxe_interior_anchor padding-top: 230px.
