About

This page lists makeables and pre-made adoptables made in the early 2000s, 2010s, and 2020s! Looking through these links is a great way to look back on the history of Neopets art and petpages. Sections for adopts of your pet are almost always called "Adopted", "Toybox", or "Clones". I highly recommend creating a section like this on your pet lookups or petpages.

Many of adoptables come with flawed code, but most of them do not. Some codes have the wrong credit link, the wrong image, or a broken image. If you're not good with HTML and want a quick fix to avoid these problems then use these codes as a template.

Right click the image and select "copy link address" to get the petpage link.

Always Do These When Grabbing Adopts

- Always copy the original code when possible. This is to preserve the hover text some adopts have.

- Link to the source petpage only, not userlookups. This is so others can find the art.

- If a url has "petpages." or "homepage" in it, these are outdated URL formats that will not load the petpage.
Make sure you put them in this format: /~PETNAMEHERE

Save Character Space (User Lookup/Pet Lookup)

- Use /~PETNAMEHERE for credit links.

- Links such as www.neopets.com/inventory.phtml can be shortened to /inventory.phtml

- Remove because it's useless.

- Rehost images to give them shorter URLs.

Remove Image Host Watermarks & Keep Images From Breaking

- Right click and select "open image in a new tab" and save the image to your computer.

- Rehost the image elsewhere and use the new image url.

- Doing so also can prevent the image from breaking. We don't know what could happen with someone else's image host. Most are lost to time eventually.

THINGS TO ALWAYS DO:
- Save makeables as soon as you find them. Image hosts are often unreliable
- Right click "open image in a new tab" to remove the image host's watermark
- Put the petpage name in the file name to keep track

- Leave the credit as written in the makeable
- Always link the makeable back to the source petpage
- Press CTRL+F to highlight specific species names on this page

THINGS TO NEVER DO:
- Never use for a contest or trophy
- Never claim the art as your own
