welcome!

Welcome to Magnet, a quality neoboard font request and premades site hosted by Elise (theleonhearted). We feature a selection of premade fonts available for your use on the neoboards as well as an option to request a custom font just for you!

Magnet is the name of a popular song performed by Vocaloids, or synthesized music programs designed to sound like human voices. The song features two lovers singing to each other about their shared passion; its lyrics inspired the first font I ever made.

This site, like the song which inspired it, strives to touch hearts—by creating fonts that are not only beautiful, but which also may be enjoyed and admired by every Neopian.

tips and hints

This page contains a number of helpful things to know when creating, editing and working with fonts.

off-centered siggies

You may have seen people with siggies that are shifted to the right or left of center. To do this, simply use pasteable spaces! Copy and paste these (　　　　　　) to the left of your siggy if you want it moved to the right, or to the right of your siggy if you want to move it to the left.

removing the siggy line

This is actually very simple to do. Just place a left-opening square bracket ( [ ) at the end of the neoHTML box, and a right-opening square bracket ( ] ) at the beginning of the siggy box. That's all it takes!

apostrophes

Normal apostrophes will not work when you type them in the NeoHTML/siggy box. Just copy and paste the symbol ′ where you want your apostrophe to go to use it.

saving space

There are many ways to save room when making your font:

1) If you use a browser other than Firefox, you can abbreviate color tags by eliminating the hashtag, the last number in the tag, or both. For example, the color #48cccc would become 48ccc. You can also use words: for example, fontc=grey.
Note: in Firefox, I believe this practice will not register the color correctly, so take note of that.

2) To make a black-colored neoHTML or siggy, use fontc=1.

3) The default font size is 2, so if you are using this size in your font, you do not have to include it in your code. Thus fonts=2c=#ccccccf=courier, for example, would become fontc=#ccccccf=courier.

4) The default fontface is Arial, so you can eliminate it from your code if you plan on using it. Thus, a neoHTML code reading fontc=#888888s=1 will give you a neoHTML of gray, size one Arial.

5) To use the fontface Times New Roman, you do not need to write out the full name. Simply writing fontf=a will change your fontface to Times.
Note: you can use any other letter of the alphabet and this will work just as well; 'a' just happens to be the one I use.

6) Use [p] in place of [br][br] to make a paragraph break.
