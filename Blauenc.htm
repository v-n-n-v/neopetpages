Hello there silent_but_crazy, Holly (silent_but_crazy) here, and welcome to the symbols page! OuO
This is a little page of common/helpful symbols that I obtained from Character Map*--all in size 4 text. The appearance of many symbols varies from browser to browser, as well. Also, they are not Webdings or Wingdings.
Can't see all the symbols? Some may not be viewable by computers that don't have Microsoft Office because some font faces that come with Office contain most of these symbols. They are SimSun, as well as any ONE of the following: MS Gothic, MS Mincho, MS PGothic, or MS PMincho. Click for info on how to obtain and install fonts!
Feel free to neomail me if you have any feedback, suggestions, criticisms, or the like! I really appreciate any suggestions or constructive criticism because I'm always looking for ways to make this site more user-friendly! ^u^

Character Map: You can get to it from the Windows Start Menu » Accessories » System Tools. It should have a little flag icon. Anyway I highly recommend checking it out; it's really cool! And you'll find many, many more symbols that aren't on this page.

MAC USERS: I now use a Mac with OS Sierra and realized it's actually even easier to access symbols than before!! To open up Characters to type symbols (and emojis), press control+command+space. Of course if you wanna see all the symbols on one page at one time you can stay on this page ;D
Before there was Character Palette which you can open through System Preferences » Personal » International » Input Menu, and click on the box next to "Character Palette."

SHAMELESS PLUG for my other font site: Here's Nebula, a font tips and tricks page. I hope it helps with your font making!
