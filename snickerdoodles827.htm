Welcome and Introduction

Hello and welcome to my Petpage!

This is a Directory of ALL past Gallery Spotlight Winners from the OLDEST to the MOST RECENT.

My goal for this page is to make it easier for those who would like to check to see what theme has already been used and chosen... and to perhaps give ideas to those who are looking for gallery ideas.

Did you know that the first Gallery Spotlight was announced on January 22, 2003 (Y5), when galleries were located in Shops?

The status of each gallery is noted in the far right column. If the gallery is still up and running (meaning it is the SAME theme that won spotlight and it still holds those items) simply click on Active in the 'Status' column to view. For those galleries that have been sold off and emptied or accts that have been deleted (labeled Inactive ) you may still view the gallery (through the Neopets link up) by clicking on the Gallery Name in the 3rd column.

To date, this directory includes over 700 galleries!

So take your time and scroll through the list, just keep in mind it is always a work in progress.

Note: You may find using the CTRL + F function helpful when searching for a particular theme.

This page was lasted Updated on March 16, 2017.

If you find any broken links or outdated pages, please Neomail Me, thanks.

If you have found this Petpage helpful, please link back to me.
