Hello and welcome to Candy Rain, a premade userlookup site by Ashe. Candy Rain offers simplistic userlookups which are fully customizable to your liking. All the userlookups are coded in greyscale so that you have the freedom to edit how you like. You can change the colors, images and more! Every style of userlookup on Candy Rain also come with a selection of images for you to use as well.

It's recommended that you know a bit of HTML/CSS if you plan to use these lookups. They're meant to be edited to your liking. However, if you like the default settings, feel free to use the lookup how it is! Before you begin browsing the userlookups, make sure to read the rules and FAQ which are located right below the userlookups. If you have any questions or comments, feel free to send me a neomail!

I put the code on my userlookup, but it's not showing properly?
Make sure you replaced the ( ) tags around "style" to (Style appears twice in the code: at the beggining of the css and at the end. Make sure you change both!)

Why do you put the style tags in ( ) instead of the normal way?
Some of my lookups use a code called "Border Radius" and petpages can't save that code inside css tags (it can only save outside css tags and in html form)
